## Install

###### Node version

node 8.11.3+
npm 6.1.0+

##### Dependecias Globales Requeridas

```npm i -g gatsby-cli```

##### install: 
```npm i```

##### development: 
```npm start```
_corre en localhost:8000_

##### build: 
```npm run build```

##### serve build: 
```npm run serve```
_corre en localhost:9000_
